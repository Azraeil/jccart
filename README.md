### 使用方法

請不要用GitHub的download，請在自己的電腦下`git clone`指令
```
git clone git@github.com:NickWarm/jccart.git
```

去`config/database.yml`修改密碼，改成你自己的MySQL密碼，若是沒用過MySQL，可以參考我寫的[]()這篇文章
